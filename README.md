# WebTracker

the goal of this repo is to show the idea of using a composition approach to build a web tracker.

Main objectives of this project:

- Having an architecture that is feature proof
- Easy to maintain
- Highly scalable
- Modular
- Simple

## Install

```shell
yarn
```

## Build

```shell
yarn build
```

## Run

```shell
yarn start
```

## Test

```shell
yarn test
```
